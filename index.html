MotW reference

  • Core mechanic
    • Roll 2d6 + the indicated rating (Cool, etc.) + “forward” or “ongoing” if you have those
    • 7-9 is a partial success; 10+ is complete success; bad things happen on 6-
  • Actual core mechanic
    1. Describe what you’re doing, and how
    2. Keeper will react
    3. React to what the Keeper just said (step 1)
  • Hunter reference: A handout, designed for new players, that summarizes the basic moves and a few other mechanics
  • An Example Fight: MotW pages 215-222, which shows an example of combat
  • Use Magic: MotW pages 195-198, with the rules for Use Magic and Ritual Magic
